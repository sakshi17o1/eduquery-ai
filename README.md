# EduQuery AI — Study Smarter 🎓git branch

An AI-powered exam preparation platform that helps students study smarter from their own notes.

🔗 **Live Demo:** [eduquery-ai-sakshi.streamlit.app](https://eduquery-ai-sakshi.streamlit.app)

## Features
- 📄 Upload PDF/Word study material
- 💬 Ask questions from your notes (RAG-based Q&A)
- 🧠 Auto MCQ generation with explanations
- 📊 Progress dashboard with score trends
- ❌ Weak topic identification

## Tech Stack
- LangChain, FAISS, Google Gemini API
- HuggingFace Embeddings
- Streamlit

## How to Run
```bash
git clone https://github.com/sakshi17o1/eduquery-ai
cd eduquery-ai
pip install -r requirements.txt
streamlit run app.py
```
